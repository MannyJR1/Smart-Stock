# Smart & Stock – Website

ระบบจำลองการจัดการคลังสินค้า พัฒนาในรูปแบบ Website จัดทำขึ้นเพื่อนำความรู้พื้นฐานด้านวิศวกรรมซอฟต์แวร์และการพัฒนาเว็บไซต์มาประยุกต์ใช้ในการสร้างระบบที่มีการคำนวณและจัดการข้อมูลจริง

## ภาพรวมของระบบ
เว็บไซต์นี้ออกแบบมาเพื่อใช้เป็นระบบหลังบ้านสำหรับจัดการร้านค้า โดยมีฟังก์ชันการทำงานที่ครอบคลุม ดังนี้:
* **Dashboard** - แผงควบคุมแสดงสรุปภาพรวมและสถิติของร้าน
* **Products** - ระบบแสดงรายการสินค้าทั้งหมด พร้อมการคัดกรองตามหมวดหมู่
* **Stock Management** - ระบบจัดการสต็อก (เพิ่มสินค้าใหม่, เติมสต็อก, ลดสต็อก และจำหน่าย)
* **Sales** - ระบบบันทึกประวัติยอดขาย พร้อมกราฟแสดงผลรายรับ
* **Movements** - ระบบตรวจสอบประวัติความเคลื่อนไหว

## 🛠 เครื่องมือและเทคโนโลยีที่ใช้
* **HTML5:** โครงสร้างของหน้าเว็บเพจ
* **CSS3:** จัดรูปแบบหน้าตา UI 
* **JavaScript :** จัดการลอจิกการคำนวณ, ระบบค้นหา, และควบคุมการเปิด-ปิด Pop-up
* **Browser Storage:** ใช้ LocalStorage และ SessionStorage ในการจำลองฐานข้อมูลเพื่อเก็บบันทึกข้อมูลสินค้าและบัญชีผู้ใช้งาน
* **Chart.js:** ใช้งานผ่าน CDN สำหรับวาดกราฟสถิติยอดขายและความเคลื่อนไหว

## โครงสร้างไฟล์และโฟลเดอร์
เพื่อความเป็นระเบียบ โปรเจกต์นี้ได้ถูกแบ่งโฟลเดอร์ตามการทำงาน อย่างชัดเจน:
* `index.html` & `index.js` → หน้าเข้าสู่ระบบและสมัครสมาชิก
* `dashboard/` → โฟลเดอร์หน้าภาพรวมร้านค้าและรายการจำหน่ายล่าสุด
* `products/` → โฟลเดอร์หน้ารวมสินค้าในรูปแบบ Grid และหน้าต่างรายละเอียดสินค้า
* `stock/` → โฟลเดอร์หน้าตารางจัดการสต็อกและระบบบันทึกการขาย
* `sales/` & `movements/` → โฟลเดอร์หน้าประวัติการทำรายการและกราฟสถิติ
* `shared/` → โฟลเดอร์เก็บไฟล์ `global.css` และ `global.js` สำหรับตั้งค่าส่วนกลาง

## ขั้นตอนการติดตั้งและใช้งาน
1. ดาวน์โหลดไฟล์โปรเจกต์ หรือแตกไฟล์ ZIP
2. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
3. หากเข้าใช้งานครั้งแรก ให้กด "สมัครสมาชิก" เพื่อสร้างบัญชีจำลอง
4. เข้าสู่ระบบ และเริ่มต้นจำลองการจัดการร้านค้าได้ทันที 
*(หมายเหตุ: ข้อมูลสินค้าและยอดขายจะถูกรีเซ็ตเมื่อปิดเบราว์เซอร์ เพื่อความสะดวกในการพรีเซนต์งาน)*

## สิ่งที่ได้เรียนรู้จากโปรเจกต์นี้
* ฝึกการออกแบบสถาปัตยกรรมเว็บไซต์แบบหลายหน้า
* ฝึกการจัดการสถานะข้อมูลด้วย JavaScript
* พัฒนาทักษะการสร้าง Interactive UI เช่น ระบบค้นหา และหน้าต่าง Modal
* เรียนรู้การนำ External Library อย่าง Chart.js มาใช้งาน
